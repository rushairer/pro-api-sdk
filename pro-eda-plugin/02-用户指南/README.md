## 🎯 快速指南

### 从扩展广场安装

1. 打开 EDA → **设置** → **扩展** → **扩展管理器**
2. 点击 **扩展广场**
3. 浏览或搜索扩展
4. 点击 **安装**

### 从本地文件安装

1. 打开 EDA → **设置** → **扩展** → **扩展管理器**
2. 点击 **导入扩展**
3. 选择 `.eext` 文件
4. 确认安装

## ⚠️ 注意事项

### 外部交互权限

某些扩展需要访问本地文件系统或网络，请仔细阅读权限说明后再授予权限。

### 扩展来源

- ✅ 官方扩展：经过审核，安全可靠
- ⚠️ 第三方扩展：请从可信来源获取

## 🆘 常见问题

**Q: 扩展安装后不显示？**
A: 检查扩展是否已启用，在扩展管理器中切换开关。

**Q: 扩展提示权限不足？**
A: 在扩展设置中开启"外部交互权限"。

**Q: 如何卸载扩展？**
A: 在扩展管理器中找到扩展，点击卸载按钮。
