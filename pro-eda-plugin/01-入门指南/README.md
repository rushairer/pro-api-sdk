## 📋 前置知识

在开始之前，您应该具备：

- ✅ 基础的 JavaScript 知识

如果您是完全的新手，建议先学习：

- [JavaScript 教程](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript)

## 🚀 快速开始

如果您已经有开发经验，可以直接：

1. **安装依赖**

```bash
npm install @jlceda/pro-api-types
```

2. **创建项目**

```bash
mkdir my-first-plugin
cd my-first-plugin
npm init -y
npm install --save-dev typescript @types/node
```

3. **查看完整教程**
   👉 [创建第一个插件](./创建第一个插件.md)
